
# CodeAtlas

> Your On-Device AI Software Architect.

CodeAtlas transforms any software project into an interactive architectural map.

Instead of reading hundreds of files, developers can understand an entire codebase visually using local AI.

## Features

- 🏗 Architecture Reconstruction
- 🌌 Project Galaxy
- 🌋 What-If Simulator
- 📖 AI Onboarding
- 🏥 Project Health Score
- 🔍 Repository Question Answering

## Built With

- React
- FastAPI
- Tree-sitter
- Qwen2.5-Coder
- ONNX Runtime
- Tauri

## Privacy First

100% On Device.
No source code leaves your machine.
